3 呼吸系统

学习笔记
作者: MingXiao

肺通气原理

肺通气动力克服阻力；气体进出肺取决于肺与外界换进的压力差

动力

  • 呼吸运动：呼吸肌收缩和舒张造成胸廓扩张和缩小的过程
    • 平静呼吸：安静状态下
      • 吸气：吸气肌收缩
      • 呼气：吸气肌舒张
      • 吸气主动，呼气被动
    • 用力呼吸：CO2↑，O2↓时，呼吸加深加快；都是主动过程
    • 胸式呼吸：肋间外肌收缩，引起胸廓变化；妊娠、腹水、肿瘤等
    • 腹式呼吸：膈肌舒缩，引起腹壁变化；胸膜炎、胸腔积水
  • 肺内压：肺泡内的压力；周期变化
    • 肺通气的直接动力是肺内压与大气压的插值
    • 人工呼吸：人工方式造成肺内压与大气压之差
  • 胸内负压：胸膜腔内压是胸内压
    • 前提：胸膜腔的密闭性
    • 始终低于大气压
    • = 肺内压 - 肺回缩力

阻力

  • 弹性阻力：在外力作用弹性组织抗变形的回位力
    • 占2/3
    • 顺应性：=1/弹性阻力
    • 静态顺应性曲线：曲线的斜率 = 顺应性
    • R↓，呼气困难，肺气肿；R↑，吸气困难，肺充血/纤维化，表面活性物质↓
    • 弹性肺阻力 = 肺表面张力（2/3） + 肺弹性纤维回缩力（1/3）
    • 肺表面活性物质
      • 本质：二棕榈酰卵磷脂
      • 合成释放：肺泡Ⅱ型细胞
      • 分布：肺泡半径越大，活性物质密度越小
      • 作用
        • 降低肺泡表面张力，调节肺泡回缩力和吸气阻力
        • 维持肺泡稳定
        • 防止肺水肿
    • 胸廓弹性阻力：方向可变
    • 总弹性阻力 = 肺弹性阻力 + 胸廓弹性阻力
    • 总顺应性 = 1/肺顺应性 + 1/胸廓顺应性
  • 非弹性阻力
    • 惯性阻力
    • 粘滞阻力
    • 气道阻力：80-90%
      • 主要产生于鼻（50%）、声门（25%）、气管和支气管（15%）

评价指标

  • 潮气量：平静时呼吸幅度0.5L
  • 肺活量：作为肺通气功能的指标
  • 时间肺活量：第一秒的用力肺活量，评价肺通气功能较好的指标

基本肺容积和肺容量

基本肺容积：不重叠的四部分

  • 补吸气量
  • 潮气量
  • 补呼气量
  • 残气量

肺容量：几项的联合气量

  • 3+4 = 功能残气量
  • 1+2 = 深吸气量
  • 1+2+3 = 肺活量
  • 1+2+3+4 = 肺总量

肺通气量与肺泡通气量

肺通气量：吸入的气体

肺泡通气量：吸入肺泡能与血液进行气体交换的气体总量

每分钟肺泡通气量 = （潮气量 - 无效腔气量） * 呼吸频率

肺换气和组织换气的过程及其影响因素

原理：依靠气体分压差进行单纯扩散

  • 分压 = 总压 * 气体容积百分比
  • 随呼吸道分级上升，氧分压↓，二氧化碳分压↑；N2，H2O无变化

肺换气的影响因素

  • 呼吸膜的厚度
    • 越厚越难换气
  • 呼吸膜的面积
    • 越大越容易
  • 通气:血流（V/Q比值）：每分钟肺泡通气量与肺血流量比值，正常值0.84
    • ↑（肺血管栓塞），产生肺泡无效腔↑
    • ↓（支气管痉挛），产生功能性动-静脉短路
    • 分布不均匀，自下而上，通气量↓，血流量↓比值↑
    • 意义：血-气平衡的衡量指标

组织换气的影响因素：也就是扩散速率的影响因素

以下全加快

  • 分压↑
  • 温度↑
  • 溶解度↑
  • 分子量↓
  • 呼吸膜扩散面积↑
  • 肺扩散容量↑
  • 呼吸膜厚度↓
  • 肺通气量↑
  • V/Q = 0.84，正常；>，↓，生理无效腔增多；<，↓，功能性A-V短路

氧和二氧化碳在血液中的运输形式

物理溶解少，化学结合多；但是得先物理溶解才能化学结合

O2：血红蛋白运输

  • Hb = 1珠蛋白 + 4血红素（Fe+4吡咯基）
  • 结合特征：反应快，可逆，不需要催化，受O分压影响；Fe+O2加氧合；1Hb结合4O2；解离曲线呈S型

CO2：

  • 碳酸盐方式，HCO3-
    • pH降低，CO2↑，肺呼出↑：通气障碍时会呼吸性酸中毒
    • pH升高，CO2↓，肺呼出↓：通气过度时会呼吸性碱中毒
  • 氨基甲酸血红蛋白结合后运输
  • CO2解离曲线接近线性曲线，不具有饱和性
  • O2与Hb结合促进CO2释放：何尔登效应

血氧饱和度

氧容量：100ml的血中，最大结合O2量

氧含量：实际

氧饱和度 = 氧含量 / 氧容量 * 100%

动脉血O2利用系数 = 释放O2容积 / O2含量 * 100%

氧解离曲线及影响因素

血氧饱和度与动脉氧分压的关系，S型

影响

  • 酸度↑，右移，促进释放
    • 波尔效应：酸度对Hb氧结合力的影响；机制：pH改变Hb构型
      • 酸度↑，Hb结合力↓，O2释放，波尔效应↑
      • 随CO2分压↑，波尔效应↑
  • 温度↑，右移，促进释放
    • 运动时促进释放

化学因素对呼吸的调节

CO2通过波尔效应影响O2与Hb的结合和释放

O2通过何尔登效应影响CO2与Hb的结合和释放

都是促进

化学感受器

  • 中枢化学感受器
    • 位于延髓腹外侧表层
    • 有效刺激是脑脊液H+浓度而非CO2
    • 胆碱能机制
  • 外周化学感受器
    • 位于颈动脉体（调节呼吸）和主动脉体（调节循环）
    • 当氧分压↓（主要），CO2分压和H+增大时传入冲动变多
    • I型胞内Ca↑

化学因素

  • CO2
    • 一定水平，维持呼吸和呼吸中枢的兴奋性
    • 分压↑，呼吸加深加快，肺通气量↑
    • 作用机制：外周次要，快速通气增强反应，不敏感；中枢主要，慢速通气增强，敏感：间接作用，游离出H+
  • O2
    • 分压↓，呼吸加深加快，肺通气量↑
    • 作用机制：外周兴奋性强，耐受性强；直接抑制中枢，通过外周的兴奋对抗
    • 严重缺氧不足以对抗，会导致呼吸障碍
  • H+
    • 不易经过血脑屏障，主要通过外周实现
    • pH↓，呼吸加深加快，肺通气量↑
    • 作用机制：外周主要，敏感性弱；中枢弱
  • 对通气效应的强弱顺序：PCO2↑ > pH >↓ PO2↓
